# Code of Conduct

## Our Pledge

We are committed to making this project a welcoming, respectful, and
harassment-free space for everyone who contributes, reports issues, reviews
code, writes documentation, or participates in project discussions.

## Expected Behavior

- Use welcoming and inclusive language.
- Be respectful of differing viewpoints and experience levels.
- Give and receive constructive feedback with care.
- Focus criticism on ideas, code, and project outcomes, not on people.
- Respect privacy, security boundaries, and responsible disclosure processes.

## Unacceptable Behavior

Unacceptable behavior includes harassment, discriminatory language, personal
attacks, intimidation, unwanted attention, publishing private information
without consent, or any behavior that would reasonably make participation feel
unsafe or hostile.

Because this project touches identity, cryptography, WebAuthn, and local-first
security flows, do not pressure others to disclose private credentials, seeds,
keys, vulnerability details, or personal identity information in public
channels.

## Reporting

If you experience or witness unacceptable behavior, please contact the project
maintainers through GitHub. For security-sensitive issues, follow the reporting
process in `SECURITY.md` instead of posting details publicly.

Reports should include enough context for maintainers to understand what
happened, while avoiding unnecessary disclosure of private information.
Maintainers will review reports in good faith and handle them as confidentially
as practical.

## Enforcement

Maintainers may take action they consider appropriate, including clarification,
moderation, temporary limits on participation, removal from project spaces, or
permanent bans for repeated or severe violations.

Anyone asked to stop unacceptable behavior is expected to comply immediately.

## Scope

This code of conduct applies in project spaces, including GitHub issues, pull
requests, discussions, documentation, examples, and any other official project
collaboration channels.
